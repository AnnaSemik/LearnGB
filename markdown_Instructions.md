## Списки
Чтобы добавить ненумерованный список, необходимо пункты выделять *. Например:

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованный список, необходимо пункты просто пронумеровать. Например:

1. Элемент 1
2. Элемент 2
3. Элемент 3
